# codex-opero v0.1.9

## English

This release focuses on making Antigravity (`agy`) usage reporting match what the CLI actually shows.

### What's Changed

- Antigravity usage now prefers live `agy /usage` output instead of stale local quota cache files.
- Antigravity is displayed as two shared quota buckets:
  - `Google`: Gemini 3.1 Pro and Gemini 3.5 Flash variants
  - `3rd Party`: Claude Opus/Sonnet 4.6 and GPT-OSS 120B
- The detail menu now lists the selectable Antigravity models under each bucket.
- Antigravity lookup failures are no longer silently hidden behind old `100%/100%` cache values.
- Provider detail sections can be collapsed and expanded.
- Collapsed/expanded provider state is persisted across app restarts and macOS reboots.
- Codex, Claude, Gemini, and Antigravity bucket rows now use a consistent detail format.
- Gemini detail groups were updated to the current Pro, Flash, and Flash Lite model families.
- Tests were expanded for Antigravity live usage parsing, current-account cache selection, and persisted collapse state.

### Notes

Antigravity live usage lookup starts the local `agy` CLI briefly and reads the `/usage` screen.  
It requires an existing Antigravity login/keyring state on the machine.

## 한국어

이번 릴리즈는 Antigravity(`agy`) 사용량 표시가 실제 CLI의 `/usage` 화면과 일치하도록 만드는 데 초점을 맞췄습니다.

### 변경 사항

- Antigravity 사용량 조회가 오래된 로컬 quota cache 파일 대신 live `agy /usage` 출력을 우선 사용합니다.
- Antigravity 사용량을 실제 공유 quota 구조에 맞춰 두 bucket으로 표시합니다.
  - `Google`: Gemini 3.1 Pro 및 Gemini 3.5 Flash 계열
  - `3rd Party`: Claude Opus/Sonnet 4.6 및 GPT-OSS 120B
- 상세 메뉴에서 각 Antigravity bucket 아래에 선택 가능한 모델 목록을 표시합니다.
- Antigravity live 조회 실패를 오래된 `100%/100%` cache 값으로 조용히 숨기지 않습니다.
- provider별 상세 영역을 접고 펼칠 수 있습니다.
- 접힘/펼침 상태는 앱 재시작 및 macOS 재부팅 후에도 유지됩니다.
- Codex, Claude, Gemini, Antigravity의 bucket 상세 표시 형식을 통일했습니다.
- Gemini 상세 그룹을 현재 Pro, Flash, Flash Lite 모델군에 맞게 업데이트했습니다.
- Antigravity live usage 파싱, 현재 계정 cache 선택, 접힘 상태 저장 테스트를 보강했습니다.

### 참고

Antigravity live usage 조회는 로컬 `agy` CLI를 짧게 실행해 `/usage` 화면을 읽는 방식입니다.  
따라서 해당 Mac에 기존 Antigravity 로그인/keyring 상태가 있어야 합니다.
