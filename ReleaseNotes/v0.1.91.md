# codex-opero v0.1.91

## English

Small bug fix release for Antigravity (`agy`) usage reporting.

### What's Changed

- Fixed Antigravity live usage parsing when `agy /usage` renders quota data through terminal redraw escape sequences.

## 한국어

Antigravity(`agy`) 사용량 표시를 위한 작은 버그 수정 릴리즈입니다.

### 변경 사항

- `agy /usage`가 터미널 화면 갱신 escape sequence로 quota를 출력할 때 Antigravity live usage 파싱이 실패하던 문제를 수정했습니다.
