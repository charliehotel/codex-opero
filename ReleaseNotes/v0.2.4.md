# codex-opero v0.2.4

## 한국어

이번 핫픽스는 새 버전 표시가 메뉴 패널 전체를 움직이던 문제를 수정합니다.

### 변경 사항

- 새 버전이 있을 때 파란색 하단 업데이트 링크만 부드럽게 펄스합니다.
- 펄스 중에도 메뉴 패널의 위치와 클릭 영역은 고정됩니다.
- macOS 동작 줄이기가 켜져 있으면 링크는 계속 정적으로 표시됩니다.

## English

This hotfix fixes an issue where the available-update indicator animated the entire menu panel.

### Changes

- Only the blue footer update link now pulses when a new version is available.
- The menu panel stays stationary and clickable while the link pulses.
- The link remains static when macOS Reduce Motion is enabled.
