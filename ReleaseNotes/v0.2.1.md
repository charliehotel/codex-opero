# codex-opero v0.2.1

## 한국어

이번 릴리즈는 Antigravity 쿼터 요약을 실제 사용 가능 상태에 맞게 수정하고, 종료된 개인용 Gemini CLI provider를 정리합니다.

### 변경 사항

- Antigravity 상단 요약은 평소 각 모델 그룹의 5시간 잔여량을 표시합니다.
- 해당 그룹의 주간 quota를 모두 소진하면 5시간 quota가 남아 있더라도 상단에 `0%`를 표시합니다.
- 상세 영역의 `[5h]`, `[7d]` 원본 값은 그대로 유지합니다.
- `[5h]` 리셋 시간을 macOS 시간 형식에 맞춘 정확한 시:분으로 표시합니다.
- 메뉴 하단에 현재 앱 버전을 표시합니다.
- 새 버전이 있으면 하단 표시를 `v0.2.1 → v0.2.2` 형태의 부드러운 펄스 링크로 바꾸고, 클릭하면 해당 GitHub Release 페이지를 엽니다.
- 업데이트 확인 주기를 마지막 성공 확인 후 24시간으로 단축하고 기존 팝업을 제거했습니다.
- 2026년 6월 18일 개인용, Google AI Pro, Google AI Ultra의 Gemini CLI 요청 처리 종료에 맞춰 독립 Gemini provider를 제거했습니다.
- 기존에 Gemini를 선택했던 사용자는 Antigravity로 자동 이전됩니다.

### 사용 조건

Antigravity quota를 확인하려면 Antigravity 앱을 실행하고 로그인해 두어야 합니다. 앱은 `agy`를 자동 실행하지 않습니다.

## English

This release aligns the Antigravity summary with actual availability and removes the retired standalone Gemini CLI provider for individual users.

### Changes

- Antigravity normally shows each model group's remaining 5-hour quota in the top summary.
- If a group's weekly quota is exhausted, the summary shows `0%` even when 5-hour quota remains.
- The detailed `[5h]` and `[7d]` source values remain unchanged.
- `[5h]` reset times now show the exact hour and minute using the macOS time format.
- The current app version is now shown at the bottom of the menu.
- When a new version is available, the footer becomes a softly pulsing `v0.2.1 → v0.2.2` link that opens the matching GitHub Release.
- Update checks now run every 24 hours after the last successful check, replacing the previous blocking prompt.
- Removed the standalone Gemini provider following the June 18, 2026 end of Gemini CLI requests for individual, Google AI Pro, and Google AI Ultra users.
- Existing Gemini selections automatically migrate to Antigravity.

### Requirement

Keep the Antigravity app running and signed in while reading its quota. codex-opero does not launch `agy` automatically.
