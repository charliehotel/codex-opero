# codex-opero v0.1.92

## English

Small bug fix release for Antigravity (`agy`) 2.0 quota parsing.

### What's Changed

- Fixed exhausted Antigravity 3rd Party quota parsing when `agy` shows only `Refreshes in ...` without an explicit remaining percentage.
- In that state, `codex-opero` now reports the bucket as `100% used` / `0% remaining` and preserves the reset timer.

## 한국어

Antigravity(`agy`) 2.0 quota 파싱을 위한 작은 버그 수정 릴리즈입니다.

### 변경 사항

- `agy`가 남은 퍼센트 없이 `Refreshes in ...`만 표시하는 Antigravity 3rd Party quota 소진 상태를 올바르게 파싱하도록 수정했습니다.
- 이 상태에서 `codex-opero`는 해당 bucket을 `100% used` / `0% remaining`으로 표시하고, 리셋 타이머도 유지합니다.
