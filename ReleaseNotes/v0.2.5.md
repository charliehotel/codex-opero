# codex-opero v0.2.5

## 한국어

이번 릴리즈는 Codex 앱이 통합 ChatGPT 앱으로 통합됨에 따라 발생하던 사용량 조회 문제를 수정합니다.

### 변경 사항

- **통합 ChatGPT 앱 지원**: 기존 `~/.codex/auth.json`에서 액세스 토큰을 로드하는 동작은 유지하되, 백엔드 응답에서 서브 윈도우 한도 정보가 반환되지 않더라도 디코딩 오류 없이 조회가 성공합니다.
- **5시간 쿼터 누락 시 유연한 처리**: 현재 5시간 쿼터(primary_window)가 한시적으로 비활성화됨에 따라, 쿼터 데이터가 누락된 경우 `--`로 안전하게 표시합니다. 추후 서비스 측에서 쿼터가 다시 복원되면 코드 수정 없이 자동으로 화면에 정상 출력됩니다.

## English

This release fixes Codex usage fetching issues due to Codex's integration into the ChatGPT desktop app.

### Changes

- **Integrated ChatGPT App Support**: Maintains reading access tokens from `~/.codex/auth.json` but prevents decoding failures when secondary limit windows are absent in the backend API response.
- **Graceful Handling of Missing 5h Quota**: Since the 5h quota is temporarily deactivated, missing quota windows now fallback to `--`. When the 5h quota is reinstated by the service, it will automatically reappear in the UI without requiring further code updates.
