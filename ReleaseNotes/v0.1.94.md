# codex-opero v0.1.94

## English

Small bug fix release for Antigravity (`agy`) 2.0 quota parsing.

### What's Changed

- Fixed Antigravity Google bucket reset parsing when terminal redraw output places the reset timer on the same row as the model name.
- The parser now evaluates the recognized model row itself, not only the following rows, and keeps model segments bounded so reset/usage data is not borrowed from another model row.

## 한국어

Antigravity(`agy`) 2.0 quota 파싱을 위한 작은 버그 수정 릴리즈입니다.

### 변경 사항

- Antigravity 터미널 화면 갱신 출력에서 Google bucket reset timer가 모델명과 같은 row에 있을 때 이를 놓치던 문제를 수정했습니다.
- 파서가 모델명 다음 줄만 보지 않고 모델명이 포함된 row 자체도 함께 평가하며, 다른 모델 row의 reset/usage 정보를 잘못 가져오지 않도록 segment 경계를 보강했습니다.
