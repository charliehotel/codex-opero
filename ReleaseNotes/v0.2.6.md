# codex-opero v0.2.6

## 한국어

이번 릴리즈는 Antigravity 무료 계정(Starter Quota)의 사용량 조회가 되지 않던 문제를 수정합니다.

### 변경 사항

- **Antigravity 무료 계정 사용량 조회 복구**: Antigravity 서버가 무료 계정에 대해 5시간 쿼터 윈도우를 반환하지 않는 방식으로 API 응답 구조를 변경함에 따라, 5h 윈도우가 없으면 사용량 전체 조회가 실패하던 문제를 수정합니다. 이제 5h 윈도우가 없는 경우 `--`로 표시하고, weekly(7d) 윈도우 사용량은 정상 조회됩니다.

## English

This release fixes Antigravity usage lookup for free accounts (Starter Quota).

### Changes

- **Restore Antigravity free-tier usage display**: The Antigravity backend changed its API response to omit the 5h quota window for free accounts, causing the entire usage fetch to fail silently. Usage now correctly falls back: the 5h slot displays `--` and the weekly (7d) window is shown with its actual consumption.
