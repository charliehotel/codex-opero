# codex-opero v0.2.2

## 한국어

이번 릴리즈는 메뉴 하단 설정 영역을 더 작고 균형 잡힌 네이티브 macOS 컨트롤로 정리합니다.

### 변경 사항

- Refresh와 Rotate 설정을 두 개의 compact action row로 재구성했습니다.
- Refresh 행의 왼쪽 버튼은 즉시 새로고침을 실행하고, 행 안에서 최근 업데이트 시각과 refresh interval을 함께 보여줍니다.
- Rotate 행의 왼쪽 버튼은 auto rotate를 켜고 끄며, 현재 상태를 `On` 또는 `Off`로 표시합니다.
- Auto rotate가 꺼져 있으면 rotate interval picker를 비활성화해 현재 동작 상태가 더 명확하게 보이도록 했습니다.
- Launch at Login은 체크박스 대신 macOS switch 스타일로 정리했습니다.
- 하단 footer는 Quit 버튼과 버전/update 표시만 남기고 중복된 Refresh Now 행을 제거했습니다.

## English

This release cleans up the lower menu settings area with a smaller, more balanced native macOS control layout.

### Changes

- Reworked Refresh and Rotate settings into two compact action rows.
- The Refresh row button now runs an immediate refresh and shows the last update time alongside the refresh interval.
- The Rotate row button toggles auto rotate and shows the current `On` or `Off` state inline.
- The rotate interval picker is disabled while auto rotate is off, making the current behavior clearer.
- Launch at Login now uses the native macOS switch style instead of a checkbox.
- The footer now keeps only Quit and the version/update indicator, removing the duplicated Refresh Now row.
