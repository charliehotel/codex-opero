# codex-opero v0.2.3

## 한국어

이번 릴리즈는 메뉴 막대와 provider 행의 compact quota 숫자를 잔여량 또는 사용량으로 전환할 수 있게 합니다.

### 변경 사항

- Display 스위치를 추가했습니다.
- 기본값은 기존과 동일하게 잔여량 표시입니다.
- Display를 Usage로 바꾸면 메뉴 막대, 선택 provider 요약, `Codex` / `Claude` / `Antigravity` 행 오른쪽 숫자가 사용량으로 표시됩니다.
- 펼친 상세 행은 이미 `N% used`라고 명시되어 있으므로 그대로 유지합니다.

## English

This release lets the compact menu bar and provider-row quota numbers switch between remaining quota and used quota.

### Changes

- Added a Display switch.
- The default remains the existing remaining-quota display.
- Switching Display to Usage makes the menu bar, selected-provider summary, and right-side `Codex` / `Claude` / `Antigravity` row numbers show used quota.
- Expanded detail rows stay unchanged because they already use explicit `N% used` labels.
